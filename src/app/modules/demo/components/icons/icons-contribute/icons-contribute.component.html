Lignes directrices de conception et de production

Vous ne voyez pas l'icône dont vous avez besoin dans la bibliothèque? Vous pouvez en ajouter! Suivez ces instructions pour garantir la cohérence visuelle et la mise en forme appropriée.

  • Toutes les icônes doivent être uniques et non redondantes avec les icônes existantes dans le système. Recherchez dans la bibliothèque le ou les mots-clés associés à la nouvelle icône proposée pour vous assurer qu'elle n'est pas déjà représentée.
  • Toutes les icônes doivent être utilisables sur toutes les plateformes et tous les appareils pris en charge.
  • Toutes les icônes doivent être compréhensibles par un public mondial d'utilisateurs, quelle que soit leur nationalité ou leur langue.

Créer une icône

  • Créez un plan de travail 24 x 24 px pour chaque icône.
  • Assurez-vous de nommer correctement les calques et les plans de travail (ces noms seront également exportés dans le code).

Exportation de SVG

  • Rendez-vous sur icomoon.io
  • Appuyez sur Import Icons
  • Ajoutez le fichier icomoon.woff
  • Ajoutez vos icônes (possibilité de récupérer ici : boxicons.com )
  • téléchargez le nouveau fichier
  • Remplacer l'ancien fichier /assets/fonts/common/icons/icomoon.woff
